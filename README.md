# 🖥️ Dell PowerEdge R630 Home Lab

## Overview
This repository documents the build, configuration, and administration 
of a Dell PowerEdge R630 enterprise server home lab. The goal of this 
lab is to develop hands-on skills in server hardware, Linux 
administration, and data center operations — targeting a 
Data Center Technician role.

---

## 🔧 Hardware Specifications

| Component | Details |
|---|---|
| Server | Dell PowerEdge R630 1U Rack Server |
| CPUs | 2x Intel Xeon E5-2640 v3 2.60GHz 8-Core |
| RAM | 32GB DDR4 ECC RDIMM |
| RAID Controller | Dell PERC H730 12Gbps |
| Network Card | Intel X520/I350 Quad Port 10Gbps |
| Power Supplies | 2x Dell 750W Platinum Redundant PSU |
| Drive Bays | 8x 2.5" SFF Hot Swap |
| Storage | 2x Dell 600GB 10K SAS 2.5" — RAID 1 (Mirrored) |
